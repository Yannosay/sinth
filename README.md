# Sinth

A declarative language that compiles to pure HTML, CSS, and JavaScript. Zero runtime, maximum control.

## 

### Install ( Recommended Way)

* `npm install -g sinth`

## 

### Quick start

* `sinth init`
* `sinth dev pages/index.sinth`




## 

### Commands

* `sinth init` — scaffold a new project
* `sinth dev` — start dev server with live reload
* `sinth build` — compile to static files
* `sinth check` — lint without output
* `sinth version` — print version
